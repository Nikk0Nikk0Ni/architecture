# Архитектура программных систем

Репозиторий содержит выполненные практические работы по курсу архитектуры программных систем. 

**Тема проекта:** Платформа для заказа услуг (аналог Profi.ru)  
**Вариант:** 4  

## 📂 Структура репозитория

* **lr1** — Проектирование архитектуры
  * Описание микросервисной архитектуры платформы (User Service, Service Catalog, Order Service).
  * Код архитектурной модели на Structurizr DSL (диаграммы System Context, Container, Dynamic).

* **lr2** — Разработка REST API сервиса
  * Реализация спроектированного API на **Python + FastAPI**.
  * Настроена система безопасности с использованием JWT-токенов (авторизация и защита эндпоинтов).
  * Написаны интеграционные тесты с помощью `pytest`.
  * Приложение полностью упаковано в Docker (`Dockerfile` + `docker-compose.yaml`).
  * Сгенерирована спецификация OpenAPI.

## 🛠 Стек технологий
* **Проектирование:** C4 Model, Structurizr DSL
* **Backend:** Python 3.10+, FastAPI, Pydantic
* **Безопасность:** JWT, passlib
* **Тестирование:** pytest, httpx
* **Инфраструктура:** Docker, Docker Compose
